# Инструкция настройки кружения для запуска unit-tests.

1. Проверьте, что на вашем компьютере установлена JRE. Проверить это можно с помощью комманды `java --version`.

2. Запусть `npm i`.

3. Глобально установить [mocha](https://github.com/mochajs/mocha), [chai](https://github.com/chaijs/chai), [ts-node](https://github.com/TypeStrong/ts-node) с помощью комманды `npm install mocha chai ts-node -g`.
